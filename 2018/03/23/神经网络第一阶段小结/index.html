神经网络第一阶段小结

发表于 2018-03-23 | 分类于 algorithm |

“神经网络是由具有适应性的简单单元组组成的广泛并行互联的网络，它的组织能够模拟生物神经系统对真实世界物体作出的交互反应”[Kohonce,1988]，初次接触神经网络，便深深的爱上了它，简直太过神奇，虽然到现在为止还没有人能证明所谓的“黑箱理论”，但是其应用早已如雨后春笋般拔地而起，并呈一发不可收拾之势迅速扩张。笔者在上完Andrew Ng教授的deep learning第一门课程之后，简直根本停不下来了……以下为一个简单的两层神经网络的公式推导和代码实现，并给出了反向传播算法的纯手工推导，纯手工的……作为第一门课程的课程笔记。如果想要搭建一个深度神经网络（DNN），使用for-loop即可完成。

神经网络反向传播算法推倒及代码实现（Python）

1，首先初始化参数

由于同一隐藏层各个神经元的计算方式相同，如果将W初始化全为0，则会导致同一隐藏层各个神经元的值相同（相当于只有一个神经元在起作用），因此达不到学习不同特征的目的，这样就有可能导致模型失效，无法收敛。因此在这里使用高斯随机数来初始化参数W，而b初始化全为0对NN的训练没有影响。

2，forward propagation

$$Z^{[1]} = W^{[1]}X+b^{[1]}$$
$$A^{[1]} = g^{[1]}(Z^{1})$$
$$Z^{[2]} = W^{[2]}A^{[1]}+b^{2}$$
$$A^{[2]} = g^{[2]}(Z^{[2]}) = \sigma(Z^{[2]})$$

3，损失函数的计算（在此使用交叉熵作为损失函数）

$$ J = -\frac{1}{m}\sum_{i = 0}^m(y^{(i)}log_2(a^{[2]{(i)}})+(1-y^{(i)})log_2(1-a^{[2]{(i)}})) $$
采用的是每个样本上的Loss，不除m也可以，只是需要更多的迭代次数来使该算法收敛，除以m能加快收敛速度。

4，backward propagation

$$dZ^{[2]} = A^{[2]} - Y$$
$$dW^{[2]} = \frac{1}{m}dZ^{[2]}A^{[1]^{T}}$$
$$db^{[2]} = \frac{1}{m}dZ^{[2]}$$
$$dZ^{[1]} = W^{[2]^{T}}dZ^{[2]}*g^{[1]^{‘}}(Z^{[1]})$$
$$dW^{[1]} = \frac{1}{m}dZ^{[1]}X^T$$
$$db^{[1]} = \frac{1}{m}dZ^{[1]}$$

6，补充backward propagation的推导

由于第2层使用的是sigmoid激活函数，故
$$ \frac{dL}{da^{[2]}} = a^{[2]}(1 - a^{[2]}) $$
而$$\frac{dL(a^{[2]}, y)}{da^{[2]}} = -\frac{y}{a^{[2]}} + \frac{1-y}{1-a^{[2]}}$$
$$\frac{dL}{dZ^{[2]}} = \frac{dL}{da^{[2]}}\frac{da^{[2]}}{dZ^{[2]}} = a^{[2]}(1 - a^{[2]})[-\frac{y}{a^{[2]}} + \frac{1-y}{1-a^{[2]}}] = a^{[2]} - y$$
故\(dZ^{[2]} = A^{[2]} - Y\)
由\(Z^{[2]} = W^{[2]}a^{[1]} + b^{[2]}\)
$$\frac{dL}{dW^{[2]}} = \frac{dL}{da^{[2]}}\frac{da^{[2]}}{dZ^{[2]}}\frac{dZ^{[2]}}{dW^{[2]}} = dZ^{[2]}\frac{dZ^{[2]}}{dW^{[2]}} = dZ^{[2]}a^{[1]^{T}}$$
故\(dW^{[2]} = \frac{1}{m}dZ^{[2]}A^{[1]^{T}}\)（\(W^{[2]}\)是向量，因此要转置）
$$\frac{dL}{db^{[2]}} = \frac{dL}{da^{[2]}}\frac{da^{[2]}}{dZ^{[2]}}\frac{dZ^{[2]}}{db^{[2]}} = dZ^{[2]}\cdot1 = dZ^{[2]}$$
即\(db^{[2]} = dZ^{[2]}\)
$$\frac{dL}{dZ^{[1]}} = \frac{dL}{da^{[2]}}\frac{da^{[2]}}{dZ^{[2]}}\frac{dZ^{[2]}}{da^{[1]}}\frac{da^{[1]}}{dZ^{[1]}} = W^{[2]^{T}}\cdot dZ^{[2]}\cdot \frac{d^{[1]}}{dZ^{[1]}}$$
$$dZ^{[1]} = W^{[2]^{T}}\cdot dZ^{[2]}\cdot g^{[1]^{‘}}(Z^{[1]})$$
\(g^{[1]}\)为第一层的激活函数，这里是tanh
$$dW^{[1]} = \frac{dL}{da^{[2]}}\frac{da^{[2]}}{dZ^{[2]}}\frac{dZ^{[2]}}{da^{[1]}}\frac{da^{[1]}}{dZ^{[1]}}\frac{dZ^{[1]}}{dW^{[1]}} = dZ^{[1]}\cdot X^T$$
同理\(db^{[1]} = dZ^{[1]}\)
总结一句话，BP反向传播的唯一原则，精髓就是链式法则。
